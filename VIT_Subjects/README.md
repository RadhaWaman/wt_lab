# VIT Pune | SDP, EDI, DT & Course Projects

A dynamic academic showcase highlighting SDP, EDI, DT, and Course Projects at Vishwakarma Institute of Technology, Pune.

## How to Open

### Option 1: Direct File Path in Browser
1. Open your web browser (Chrome, Firefox, Safari, Edge, etc.)
2. Press `Ctrl + O` (Windows/Linux) or `Cmd + O` (Mac)
3. Navigate to the project folder and select `index.html`
4. Click "Open"

### Option 2: File Explorer/Finder
1. Open your file explorer (Windows) or Finder (Mac)
2. Navigate to the project directory
3. Double-click on `index.html`
4. The file will open in your default browser

### Option 3: From Terminal/Command Line

Navigate to the project directory first, then run:

**Windows (PowerShell):**
```powershell
cd "path\to\Syllabus_Assignment2"
Start-Process "index.html"
```

Or simply:
```powershell
Start-Process index.html
```

**Windows (Command Prompt):**
```cmd
cd path\to\Syllabus_Assignment2
start index.html
```

**Mac (Terminal):**
```bash
cd /path/to/Syllabus_Assignment2
open index.html
```

**Linux (Terminal):**
```bash
cd /path/to/Syllabus_Assignment2
xdg-open index.html
```

## Project Structure

- `index.html` - Main HTML file
- `styles.css` - Stylesheet for the project
- `script.js` - JavaScript functionality

## Requirements

- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No server or additional setup required

## Features

- Interactive academic components explorer
- Student Impact Meter
- Bootstrap 5.3.3 responsive design
- Dynamic pill navigation system
- Mobile-friendly interface

## Notes

- This project uses Bootstrap CDN, so an **internet connection** is required to load Bootstrap styles
- All files must be in the same directory for proper linking
- JavaScript and CSS are locally referenced, so they work without a server
